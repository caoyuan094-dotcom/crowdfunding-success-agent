# 评分规则与成功率诊断口径

## 总分与评级

总分 100 分，六大维度：

| 维度 | 分值 |
|---|---:|
| 产品差异化与海外市场适配 | 17 |
| 样机成熟度与交付风险 | 17 |
| 定价档位与盈利模型 | 16 |
| 素材与转化内容储备 | 16 |
| 流量投放与预热资源 | 17 |
| 团队与合规兜底 | 17 |

评级：

| 分数 | 等级 | 判断 |
|---:|---|---|
| 90-100 | S 级 | 强推，具备高潜力 |
| 75-89 | A 级 | 可推进，需局部优化 |
| 60-74 | B 级 | 能做但需要明显整改 |
| 0-59 | C 级 | 不建议直接启动 |

成功率表达：

| 评级 | 成功率表达 |
|---|---|
| S 级 | 高成功率候选项目 |
| A 级 | 中高成功率项目 |
| B 级 | 中低成功率项目，必须整改后推进 |
| C 级 | 低成功率项目，不建议直接启动 |

## 维度 1：产品差异化与海外市场适配（17 分）

子项：

- 海外真实高频痛点：4 分
- 同类众筹项目饱和度与竞品压力：3 分
- 独家专利、结构、外观、材料或体验创新：4 分
- 欧美消费习惯与使用场景匹配：3 分
- 品类众筹天然适配度：3 分

判断：

- 14-17：爆款潜力
- 9-13：合格可做
- 0-8：市场竞争力不足

常见扣分：

- 只是国内卖点翻译，没有海外痛点。
- 与亚马逊成熟商品差异不明显。
- 功能多但购买理由不集中。
- 体积、重量、电压、使用场景不适合欧美用户。

## 维度 2：样机成熟度与交付风险（17 分）

子项：

- 可演示工程样机状态：5 分
- BOM、开模、MOQ、供应链预算真实性：4 分
- 认证计划与进度：3 分
- 工厂履约经验与产能：3 分
- 生产周期与交付缓冲：2 分

判断：

- 14-17：交付低风险
- 9-13：中等风险
- 0-8：交付高风险

常见扣分：

- 只有渲染图，没有可演示样机。
- BOM 或 MOQ 未确认。
- 电子类产品没有 FCC / CE / UL 等认证计划。
- 工厂无众筹或海外订单履约经验。
- 交期过紧，没有延期缓冲。

## 维度 3：定价档位与盈利模型（16 分）

子项：

- Early Bird、Regular、Bundle、Add-on 档位设计：4 分
- 毛利率与众筹安全线：4 分
- 筹款目标合理性：3 分
- 运费、关税、平台费、支付费测算：3 分
- 配件加购与客单价提升能力：2 分

判断：

- 13-16：盈利健康
- 8-12：利润偏薄
- 0-7：亏损风险高

常见扣分：

- 毛利率低于 45%。
- 未测算国际运费、税费、平台手续费、广告成本。
- 早鸟价没有吸引力。
- 筹款目标过高，影响达标速度。
- 无套装和加购设计。

## 维度 4：素材与转化内容储备（16 分）

子项：

- 主视频完整度与前 15 秒吸引力：4 分
- 详情页结构与首屏转化：4 分
- 场景、实测、对比、开箱、耐久测试素材：4 分
- 媒体、KOL、测评、工厂、创始人背书：2 分
- FAQ、风险说明、交付说明完整度：2 分

判断：

- 13-16：转化素材充足
- 8-12：需要明显补素材
- 0-7：不具备上线内容条件

常见扣分：

- 只有棚拍图或渲染图。
- 视频更像国内招商片，不像众筹转化片。
- 没有痛点、对比、实测、场景。
- 页面没有解决信任和交付疑虑。

## 维度 5：流量投放与预热资源（17 分）

子项：

- 邮件池、私域、预热页：4 分
- KOL、媒体、社群资源：4 分
- 广告预算与投放计划：4 分
- 上线 72 小时冲量方案：3 分
- 拉长期与二次传播计划：2 分

判断：

- 14-17：流量闭环较完整
- 9-13：流量勉强支撑
- 0-8：冷启动风险高

常见扣分：

- 无邮件池、无预热页。
- 无 KOL 或媒体计划。
- 广告预算低于项目所需。
- 没有上线 72 小时冲量节奏。
- 只期待平台自然流量。

## 维度 6：团队与合规兜底（17 分）

子项：

- 创始人或操盘团队众筹经验：4 分
- 收款、税务、海外售后方案：3 分
- 平台规则与禁售风险：3 分
- 知识产权、专利、商标风险：3 分
- 售后赔付、客诉、退款机制：2 分
- 客户配合度与决策效率：2 分

判断：

- 14-17：团队和合规较稳
- 9-13：基础合规
- 0-8：封号、维权、售后风险高

常见扣分：

- 团队没有众筹经验，也无专业服务商介入。
- 收款、税务、售后不清楚。
- 疑似侵权、仿牌、擦边宣传。
- 售后赔付机制缺失。

## 硬风险与评级限制

| 情况 | 处理 |
|---|---|
| 只有渲染图、无样机 | 最高 B 级 |
| 毛利率低于 30% | 最高 B 级，严重时 C 级 |
| 电子类无认证计划 | 最高 B 级 |
| 疑似侵权、仿牌、禁售品 | 直接 C 级 |
| 无预算、无私域、无 KOL、无预热计划 | 流量维度不得高于 8 分 |
| 无 BOM、无 MOQ、无交付计划 | 样机成熟度与交付风险不得高于 10 分 |

## 资料缺失评分原则

- 缺资料不代表一定不能做，但必须降低结论确定性。
- 对缺失关键资料的项目，采用保守评分。
- 不得编造竞品数据、广告数据、认证状态、工厂履约案例。
- 报告中必须说明缺失资料影响了哪些分数。
